Styles
======
Grace à ces styles il est possible de changer très facilement l'apparence des diagrammes. De tels styles sont entre autre nécessaires par exemple pour les diagrammes de cas d'utilisation.

Installation
------------
Pour installer ces styles, il suffit de copier ces différents fichiers dans le répertoire  $WORKSPACE/$PROJET/data/.config/styles. Ce répertoire contient déjà des fichiers comme "default.style", "extern.style", ...

ATTENTION: Les styles sont en cascades et certains styles peuvent donc en nécessiter d'autres. Il est donc préférable de copier tous les styles de ce répertoire.

Utilisation
-----------
Une fois les styles copié et modelio lancé, un style peut être appliqué à un diagramme entier (ou éventuellement juste à l'élément sélectionné) en utilisant dans la palette "Symbol" le menu déroulant le plus à droite.
