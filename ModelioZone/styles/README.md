Styles
======
Grace à ces styles il est possible de changer très facilement l'apparence des diagrammes. De tels styles sont entre autre nécessaires par exemple pour les diagrammes de cas d'utilisation.

Installation
------------
Télécharger le fichier "stylesXXX.zip" et le décompresser. Pour installer ces styles, il suffit ensuite de copier les différents fichiers ".style" dans le répertoire  $WORKSPACE/$PROJET/data/.config/styles où $WORKSPACE et $PROJET sont respectivement les emplacements de l'espace de travail et le nom du projet. Le répertoire "styles" doit déjà contenir déjà des fichiers comme "default.style", "extern.style", ...

ATTENTION: Les styles sont en cascades et certains styles peuvent donc en nécessiter d'autres. Il est donc préférable de copier TOUS les styles de ce répertoire.

Utilisation
-----------
Une fois les styles copiés et Modelio (re)lancé (fermer et ouvrir le projet de nouveau devrait être suffisant), un style peut être appliqué à un diagramme entier (ou éventuellement juste à l'élément sélectionné) en utilisant dans la palette "Symbol" le menu déroulant le plus à droite. Si un élément dans le diagramme est selectionné, le style s'appliquera uniquement sur cet élémént, sinon le style sera appliqué au diagramme. Il est ainsi possible de "surcharger" le style d'un diagramme pour un élément particulier. Il est aussi possible d'utiliser les multiples options de la palette "Symbol" pour des modifications locales à un diagramme.
